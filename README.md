洛克王国 伤害计算器 — 阶段一

说明：
- 单文件前端实现，基于 Vue 3 CDN + PapaParse，用于解析 CSV 并展示六维数值。
- 支持：从 ./宠物种族值全览.csv 自动加载（若将 CSV 放在项目根或 public），或通过“上传 CSV”按钮上传本地文件。
- 支持 IV (0/60) 与 性格修正 (0/0.2) 的选择；可选择展开显示四种配置对比或仅显示当前配置。
- 可搜索（名称或属性）、导出当前视图为 JSON、复制当前配置 JSON。

快速运行：
1. 将仓库克隆到本地（或把 CSV 放到项目根）：确保文件名为 宠物种族值全览.csv
2. 用浏览器直接打开 index.html（无需构建工具）或使用本地静态服务器（推荐）。
